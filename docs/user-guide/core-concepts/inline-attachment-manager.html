    Inline Attachment Manager

    Depending on your configuration, you can upload several attachments. A default installation of MixERP allow you to upload :

    • Image Files (JPEG, GIF, PNG)
    • Word Documents (DOC, DOCX)
    • Spreadsheets (CSV, TSV, XLS, XLSX)
    • PDF documents
